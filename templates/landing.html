About SSTS

SSTS is an innovative traffic monitoring system developed for the Astranauts by Astra competition. Our system utilizes advanced computer vision and deep learning technologies to provide real-time traffic analysis and improve road safety in Demak Regency. Our system offers:

  • Real-time vehicle detection and tracking using YOLO and DeepSORT
  • Intelligent traffic congestion monitoring and analysis
  • Automated accident detection and alert system
  • Road condition monitoring with pothole detection
  • Multi-stream support for comprehensive coverage
  • Real-time traffic status updates and statistics
